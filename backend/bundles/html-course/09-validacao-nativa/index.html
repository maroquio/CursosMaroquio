Atributos de Validação

required

Campo obrigatório. O formulário não envia sem preencher.

minlength / maxlength

Define quantidade mínima e máxima de caracteres.

min / max

Define valores numéricos mínimo e máximo aceitos.

pattern

Expressão regular que o valor deve corresponder.

placeholder

Texto de exemplo exibido quando o campo está vazio.

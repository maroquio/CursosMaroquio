Elementos Interativos HTML5

Interaja com cada elemento para vê-lo em ação.

<dialog>

Modal nativo do HTML — sem JavaScript extra para abrir/fechar.

Modal Nativa

Este é um <dialog> nativo do HTML5. Tem backdrop, fecha com Esc, e gerencia foco automaticamente.

 
<dialog id="myDialog">
  <h3>Título</h3>
  <p>Conteúdo...</p>
  <form method="dialog">
    <button>Fechar</button>
  </form>
</dialog>

<button onclick="myDialog.showModal()">
  Abrir
</button>

<details> / <summary>

Accordion nativo — abre e fecha sem JavaScript.

O que é HTML5?

HTML5 é a versão mais recente do HTML, com novos elementos semânticos, APIs e multimídia nativa.

Preciso de JavaScript para <details>?

Não! O comportamento de abrir/fechar é nativo do navegador. Zero JavaScript necessário.

Posso estilizar com CSS?

Sim! Você pode estilizar <summary> e o conteúdo com CSS normal. O marcador (▶) pode ser customizado com ::marker.

<details>
  <summary>Pergunta?</summary>
  <p>Resposta aqui.</p>
</details>
